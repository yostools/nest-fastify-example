# Example Server

Example server for NestJS and fastify.

## Description

The server is based on the [NestJS](https://github.com/nestjs/nest) framework and [fastify](https://www.fastify.io/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
